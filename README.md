# FyncX 

#### 基于TCP的文件同步工具

#### 一、特点

1. 同步实现
   - [x] 单向同步
   - [ ] 双向同步

2. 支持格式：IMG、ZIP、EXE...

#### 二、开发框架：QT5/Qwidget

#### 三、INSTALL

  源码编译：

```git
git clone https://github.com/sangko-dgq/FSyncX.git
```

 安装包

[Releases](https://github.com/sangko-dgq/FSyncX/releases/tag/FSyncX_beta)
