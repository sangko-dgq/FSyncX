#### TCP based file synchronization tool

#### 一、Features

1. Synchronous direction support
   - [x] Unidirectional synchronization
   - [ ] Bidirectional synchronization
2. File directory support
   - [x] On Root directory
   - [ ] Multi tier subdirectory
